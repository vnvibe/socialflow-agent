============================================
  SocialFlow Agent v1.0 - Huong dan cai dat
============================================

Yeu cau:
  - Node.js 18+ (tai: https://nodejs.org)
  - Windows 10/11, macOS, hoac Linux

--------------------------------------------
  BUOC 1: Cai dat
--------------------------------------------
  Windows:  Click doi vao setup.bat
  Mac/Linux: Mo Terminal, chay: bash setup.sh

  Script se tu dong:
  - Cai thu vien (npm install)
  - Cai trinh duyet Chromium
  - Tao file .env

--------------------------------------------
  BUOC 2: Dien thong tin .env
--------------------------------------------
  Mo file .env bang Notepad (hoac text editor bat ky).
  Dien 2 truong bat buoc:

  SUPABASE_SERVICE_ROLE_KEY=...  (lay tu Supabase Dashboard)
  SUPABASE_ANON_KEY=...          (lay tu Supabase Dashboard)

  Cac truong khac da co san, khong can sua.

--------------------------------------------
  BUOC 3: Chay Agent
--------------------------------------------
  Cach 1: Click doi SocialFlowAgent.exe (co giao dien)
  Cach 2: Mo Terminal, chay: node agent.js (khong can giao dien)

--------------------------------------------
  Luu y
--------------------------------------------
  - Lan dau chay se mat vai phut de tai trinh duyet
  - Agent se tu dong nhan job tu server va thuc thi
  - Dang nhap bang tai khoan SocialFlow (email/password)
  - Khong tat trinh duyet khi Agent dang chay
  - File debug/ chua screenshot loi (neu co)

  Ho tro: lien he admin SocialFlow
